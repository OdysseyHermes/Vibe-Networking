Hub 03
Reddit Vibe Coding Community

Fastest-growing developer community on Reddit. The term "vibe coding" was coined by Andrej Karpathy (Feb 2025) to describe AI-guided development by feel rather than syntax. r/vibecoding grew from near-zero to 89K+ members in under one year.

Core Links
Primary Subreddit: reddit.com/r/vibecoding 89K+ members
GitHub Org: github.com/Vibe-Coding-Community
Substack Newsletter: vibecodingcommunity.substack.com
Tool Guide (Reddit picks 2026): aitooldiscovery.com/guides/vibe-coding-reddit
Location / Format
🌐 Fully Online / Async 🌍 Global membership 📲 Reddit threads
Community Structure
VCC Education — courses, webinars, training on AI coding
VCC Agency — custom AI product dev for SMBs
VCC Crypto — open-source rewards via VCC token (TON Blockchain)
Top tools discussed: Claude Code, Cursor, Lovable, Bolt.new, Emergent
Evolution: "look what I made" → production readiness debates
